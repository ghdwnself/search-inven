# 재고 조회 시스템 사용 가이드

## 🖥️ 서버 PC (관리자용)

### 1. 초기 설정 (최초 1회만)

#### 방화벽 설정
PowerShell을 **관리자 권한**으로 실행 후:

```powershell
cd "c:\Users\james\OneDrive\Desktop\Project\inven"
.\setup-firewall.ps1
```

#### 내 PC의 IP 주소 확인
```powershell
ipconfig
```

**IPv4 주소**를 확인하세요 (예: `192.168.1.100`)

### 2. 서버 실행

```powershell
cd "c:\Users\james\OneDrive\Desktop\Project\inven"
npm run dev
```

서버가 시작되면 다음과 같이 표시됩니다:
```
🚀 서버가 http://localhost:5000 에서 실행 중입니다.
📡 네트워크 접속: http://<내부IP>:5000

➜  Local:   http://localhost:3000/
➜  Network: http://192.168.1.100:3000/
```

### 3. 직원들에게 공유할 주소

**Network 주소**를 직원들에게 알려주세요:
```
http://192.168.1.100:3000
```

## 👥 직원 PC (사용자용)

### 접속 방법

1. 웹 브라우저 열기 (Chrome, Edge 등)
2. 주소창에 입력: `http://192.168.1.100:3000` (실제 IP로 변경)
3. 즐겨찾기/북마크 추가 권장

### 사용 방법

#### 재고 조회
1. 왼쪽 사이드바에서 **"재고 조회"** 클릭
2. 브랜드 선택 (선택사항)
3. 검색어 입력 (제품명 또는 SKU)
4. 검색 버튼 클릭

#### 다중 검색
1. 왼쪽 사이드바에서 **"다중 검색"** 클릭
2. 엑셀에서 SKU 복사
3. 텍스트 영역에 붙여넣기
4. 검색 버튼 클릭

## ⚠️ 주의사항

### 서버 PC
- 서버 PC는 항상 켜져 있어야 함
- 터미널 창을 닫으면 안 됨
- 네트워크 연결 유지 필요

### 직원 PC
- 같은 사무실 네트워크에 연결되어야 함
- VPN 사용 시 네트워크 설정 확인 필요
- 브라우저 캐시로 빠른 접속 가능

---

## 🔧 문제 해결

### "사이트에 연결할 수 없음" 오류
1. 서버 PC의 터미널이 실행 중인지 확인
2. 같은 네트워크에 있는지 확인
3. IP 주소가 변경되지 않았는지 확인
4. 방화벽 설정 확인

### 느린 속도
1. 브라우저 캐시 삭제
2. 서버 PC 재시작
3. 네트워크 연결 상태 확인

### 이미지가 안 보임
1. 페이지 새로고침 (F5)
2. Google Drive 권한 확인
3. 서버 로그 확인
